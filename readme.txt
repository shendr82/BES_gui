Welcome to BES GUI readme file!

 -- Insallation guide for BES GUI --

You can download Modul BES GUI from GitHub:
https://github.com/shendr82/BES_gui.git

install python3

Packages to install:
- numpy
- pyqt5
- matplotlib
- scipy

Comands that work on Linux scientific:
  sudo yum install python3-numpy
  sudo yum install python3-qt5
  sudo yum install python3-devel
  sudo pip3 install matplotlib
  sudo pip3 install scipy


To start BES GUI please run file:
BES_gui/bes_gui.py
